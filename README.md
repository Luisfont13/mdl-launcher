# MDL Launcher

MDL Launcher es un lanzador personalizado para Minecraft que permite gestionar múltiples instalaciones con trazabilidad maker, soporte offline y branding ético.

Este proyecto es un **fork de [PrismLauncher](https://github.com/PrismLauncher/PrismLauncher)**, adaptado para entornos educativos, accesibles y éticamente bifurcados. Se ha reestructurado visualmente y funcionalmente para reflejar una identidad propia, manteniendo respeto por el trabajo original.

## Características

- Soporte para cuentas offline sin autenticación Microsoft
- Modularidad multiplataforma (Linux, Windows, macOS)
- Branding visual propio y glosario técnico-cultural
- Scripts de generación de íconos multiplataforma
- Documentación orientada a makers y entornos educativos
